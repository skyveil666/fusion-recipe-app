# Project: Fusion Recipe App

## Project Type
This is a **React Native / Expo** mobile app project. There is no web browser frontend.

## Preview Tools
**Do NOT use preview_* tools or require browser preview for this project.**
Browser preview is not applicable here — this is a mobile app that runs on Android/iOS via Expo Go or EAS build.

After editing code, do not verify via browser. Instead:
- Confirm the code changes are correct by reading the relevant files
- Note that the app can be tested on device via Expo Go or an EAS build

## Project Structure
- `expo-app/` — React Native / Expo mobile app (main app)
- `server/` — Express.js API server (deployed on Railway)
- `client/` — Web recipe sharing site (separate from the mobile app)

## Key Tech
- Expo SDK + React Native for mobile
- Gemini 2.5 Flash for recipe generation
- Stability AI for food image generation
- Wikipedia API for ingredient photos
- EAS Build for Android APK builds
- Railway for server deployment
